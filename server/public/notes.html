Help & Information

About Clinical AI Tools

This system provides AI-powered clinical decision support tools including ECMO-SVC scoring and STS risk assessment. It is designed to assist healthcare providers in their decision-making process.

How to Use

  1. Select your desired tool (ECMO-SVC Score or STS Risk Score)
  2. Enter patient history in the first text box
  3. Enter physical examination findings in the second text box
  4. Click "Generate Recommendations" to receive AI analysis
  5. Review the recommendations in the Analysis Results tab
  6. Print or download the report as needed

Privacy & Security

This system is HIPAA compliant. All data is encrypted and processed securely. No patient information is stored permanently.
